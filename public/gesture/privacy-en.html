What information we collect

You can access Micro Gesture anonymously, and Micro Gesture does not collect any personal information. Micro Gesture is a battery information viewer and battery life optimizer, so it requires a number of permissions, which are explained below.

  1. Data storage: All configuration data of Micro Gesture is saved locally and will not be uploaded to the server, and when using the export function, it will only be saved to the storage location specified by the user.
  2. Camera: When using the flashlight function of Micro Gesture, camera permission is required, which is a limitation of Android.
  3. Read phone status: When using the disable gesture function during a call, it will need to read the phone status to detect if it is currently in a call, Micro Gesture will not save this status and will not upload it.
  4. Network access: Micro Gesture only requires network verification when activated, but not network access at other times.
  5. Vibrate access: Micro Gesture needs vibrate permission when prompting user.
