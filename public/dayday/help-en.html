How do I use an Unscheduled task?

Unscheduled task is removed from dayday 1.9.0, before that:

  • Unscheduled tasks refer to tasks that are similar to three times a week, but not fixed on a certain day. Such tasks cannot be daily tasks, and there are intervals, such as every Sunday, or every three days.
  • Unscheduled tasks can be punched at any time on the homepage. In fact, it is calculated on the next effective day, and there is no data on the invalid days.
  • Under the history page and the task details page, you cannot modify the data of the invalid days, but you can modify the data of the effective day.

For example, if you want to run 3 times a week. You can create a task that repeats every Sunday, type is counted, and is set to be Unscheduled. Then, from Monday to Saturday, you can punch at any time on the homepage. Actually, it is added to Sunday, and there is no data from Monday to Saturday. On the history page, it is forbidden to modify the data from Monday to Saturday. If you ran on Tuesday, but forgot to punch, you can punch directly on Wednesday.

How do I set up webdav synchronization?

Requires a cloud that supports webdav. Let's take the jianguoyun as an example.

  1. Download and register jianguoyun, and log in.
  2. Enter the settings page and see that there is a third-party application management.
  3. Click Add to add an app. The app name is filled in and it will automatically generate a password.
  4. Copy the server, account, and password to the cloud sync option in the dayday settings. Then click to start syncing.
  5. If you have never configured cloud synchronization before, you will automatically sync to the cloud in the future. If you have configured it before and generated data on the cloud, you will be prompted to overwrite the cloud data. Please choose carefully.
