What information we collect

You can access BatteryGuard anonymously, and BatteryGuard does not collect any personal information. BatteryGuard is a battery information viewer and battery life optimizer, so it requires a number of permissions, which are explained below.

  1. Foreground service: BatteryGuard detects power in the background, in order to prevent the background from being cleaned by the system, so you need to apply for foreground service permission.
  2. Network access: BatteryGuard only requires network verification when activated, but not network access at other times.
  3. Vibrate access: BatteryGuard needs vibrate permission when prompting the user's battery level.
