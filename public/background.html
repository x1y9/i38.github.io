For Android applications that need to retain the background, the method of setting the background is closely related to the model, here for the common manufacturer models are explained, note that even the same manufacturer, due to different system versions may have a small difference in the settings, please refer to carefully, in addition, if the phone also installed a third-party background management app, you also need to add a whitelist in these apps, here is not detailed.

Huawei / Honor EMUI

  1. Pull down the corresponding app on the recent tasks, and the lock symbol will appear.
  2. Settings >> Apps >> App Launch Management >> Turn off the automatic management of the corresponding app, and turn on the three switches of manual management (Allow Self Launch, Allow Associated Launch, Allow Background Activity).
  3. Settings >> Battery >> Disable power saving and super power saving modes.

Xiaomi MIUI

  1. Lock the corresponding application on the recent tasksn.
  2. Long press the corresponding card in the recent tasks, enter the application settings, turn on the self-start switch, and set the power saving policy to unlimited.

Samsung One UI

  1. Lock the corresponding application on the recent tasks.
  2. Smart Manager >> Automatically run applications >> Turn on the switch.
  3. Smart Manager >> Battery >> Find the corresponding app >> Turn off the switch.

Meizu Flyme

  1. Long press on the recent tasks screen to lock the corresponding app.
  2. Phone Manager >> Permission Management >> Background Management >> Find the corresponding app and check Allow background.

Oppo ColorOS

  1. Pull down the recent tasks card to lock the corresponding app.
  2. Settings >> Application management >> Self-launch management >> Find the corresponding app and turn on the switch.
  3. Settings >> Battery >> Consumption protection >> Find the corresponding app >> Turn off the background freeze and auto-optimization.
  4. Security center >> Pure background >> Add apps >> Find the corresponding app and confirm.

Vivo Funtouch OS

  1. Pull down the recent tasks card to add lock to the corresponding app.
  2. Settings >> App Management >> Permission Management >> Self-launch >> Find the corresponding app, and turn on the switch.
  3. Settings >> Battery >> High power consumption in the background >> Find the corresponding app >> Turn on the permission.

One Plus

  1. Add a lock to the corresponding app on the recent tasks.
  2. Settings >> Battery >> Power Saving Optimization >> Find the corresponding app >> Set to no optimization.
  3. Settings >> Battery >> Power Saving Optimization >> Select Advanced Optimization in the upper right corner >> Disable Sleep Standby Optimization and Smart Battery Management.
  4. Settings >> System >> RAM Boost startup acceleration >> turn off the switch.
